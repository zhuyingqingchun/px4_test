# 重复启停验证记录

建议至少做 3~5 轮完整验证，验证“不是碰巧能飞一次”，而是“每次都能飞”。

## 1. 建议验证场景

- 首次启动后能飞
- `stop.sh` 后再次启动能飞
- `restart.sh` 后能飞
- 关闭终端重新开后能飞
- 第二天重新启动仍能飞

## 3. 阻塞级 / 非阻塞级分类

### 阻塞级

满足任意一项即归为阻塞级：

- 无法起飞
- Offboard 进入失败
- 模式切换失败
- 明显控制抖动
- 第二次启动失败

### 非阻塞级

满足以下特征可先归为非阻塞级：

- 仅在启动阶段短暂出现
- 不影响解锁 / 起飞 / 悬停 / 降落
- 不影响第二次运行
- 不影响轨迹飞行验证
