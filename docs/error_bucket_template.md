# 报错分桶模板

目标：把日志中的红字从“看起来很多”，整理成“哪些要立刻修，哪些先观察”。

## 1. 阻塞级报错

满足任意一项，建议直接归为阻塞级：

- 导致无法进入 Offboard
- 导致无法解锁 / 起飞 / 降落
- 导致控制明显异常
- 导致轨迹无法执行完成
- 导致二次启动失败
- 导致节点退出、topic 中断、关键进程缺失

## 2. 非阻塞级报错

满足以下特征，可先归为非阻塞级：

- 仅在启动阶段短暂出现
- 不影响 Offboard、解锁、起飞、悬停、降落
- 不影响轨迹执行
- 不影响第二次运行
